# 課題内容

 - GitHubアカウントの新規作成
 - リポジトリ・ブランチ作成
 - 課題内容や感想をMarkdownで記載し、プルリクエストを発行
 

 ## 学び
 
 1. Markdownは見出し、段落などの構造を持つ文章を書くためのファイル形式。
       - HTMLよりもシンプルな記述ルール。
       
 2. VScodeやcloud9にもMarkdownのビューワー機能がついている。
       - command + K + Vで表示可能。　(**macOSの場合**)　
      
 3. 「git status」と入力すると、ファイルの追加や修正など、現在のローカルリポジトリがどんな状況か確認できる。
